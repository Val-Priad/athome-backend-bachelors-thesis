# AtHome project

## Installation

Necessary env variables are:

```
APP_BASE_URL=
MEDIA_BASE_URL=
RESEND_API_KEY=
JWT_SECRET_KEY=

RATE_LIMIT_STORAGE_URI=
RATE_LIMIT_TEST_STORAGE_URI=memory://

POSTGRES_PORT=
POSTGRES_USER=
POSTGRES_PASSWORD=

DATABASE_URL=
TEST_DATABASE_URL=

REDIS_PORT=
REDIS_URL=

S3_BUCKET_NAME=
S3_REGION=
S3_ACCESS_KEY_ID=
S3_SECRET_ACCESS_KEY=
S3_PRESIGNED_URL_TTL_SECONDS=300
```
